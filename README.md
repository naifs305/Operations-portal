# بوابة إدارة عمليات التدريب

بوابة إلكترونية مؤسسية تجمع جميع المنصات والأنظمة المتعلقة بإدارة عمليات التدريب في مكان واحد.

## المميزات
- تصميم عربي RTL
- هوية بصرية متوافقة مع ألوان جامعة نايف
- شبكة بطاقات متجاوبة
- لوحة إدارة محلية داخل المتصفح
- استيراد وتصدير البيانات بصيغة JSON
- جاهزة للبناء كنسخة static

## التشغيل
```bash
npm install
npm run dev
npm run build
```

## النشر على GitHub Pages
1. شغّل `npm run build`
2. ارفع محتوى مجلد `out` إلى الاستضافة أو GitHub Pages

## ملاحظة
لوحة الإدارة في هذه النسخة تعمل محليًا داخل المتصفح وتخزن البيانات في Local Storage مع خيار النسخ الاحتياطي عبر JSON.
